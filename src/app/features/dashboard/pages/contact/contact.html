Investigación de Tesis Académica

El sistema de software y hardware SIIT 4.0 (Sistema de Invernadero Inteligente con Tecnología 4.0) ha sido concebido y estructurado como un proyecto de investigación científica para optar el título profesional de Ingeniería. Su núcleo radica en la fusión de metodologías ágiles, computación en la nube y automatización perimetral agrícola.

Objetivo General del Proyecto

Diseñar e implementar una arquitectura escalable de Internet de las Cosas (IoT) orientada a la agricultura vertical protegida, optimizando el consumo hídrico, controlando las variables climáticas críticas en tiempo real y garantizando un aprovisionamiento eficiente mediante fuentes de energía renovable.

Arquitectura Tecnológica del Proyecto

Hardware & Firmware IoT

Nodos controladores basados en microcontroladores ESP32, sensores volumétricos de humedad de suelo, termistores ambientales, almacenamiento de contingencia offline (tarjetas SD) y actuadores de válvulas solenoides acoplados a sistemas solares autónomos.

Software, API & Cloud

Plataforma modular construida en el Frontend con Angular + Tailwind CSS, Backend API de alta velocidad programado sobre FastAPI (Python), orquestación mediante Docker Compose e infraestructura de despliegue en la nube.
